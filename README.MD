### django涉及到的基础

- 自定义模板tag
- HTML模板的继承
- Django自带admin的基础使用
- 数据model的使用
- django-restframework对rest接口的使用
- Django2版本上对url规则的定义
- 借助layui前端框架进行布局
- 自定义异常(Error)返回规范
- 引用editor.md渲染前端Markdown信息流
- admin中使用django-mdeditor编写文章
- 数据调用模式：ajax请求和模板渲染
- django-validator库的使用(强烈推荐，已修改为兼容Python3)
